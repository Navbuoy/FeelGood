Feel Good Events Pack (BETA)
Created for HBS' Battletech Tabletop video-game. Exclusively for Battletech Tactics (BEX CAC mod - compatible with both TheBigDeal mod and Expension Pack)
The events are spread out from the dates of 3031 - 3055.

Just unzip the file and throw the FeelGood folder in your Battletech Mods folder and launch your game.
